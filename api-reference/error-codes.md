# Error Codes

The API error schema can be found [here](types-glossary.md#error)

e.g.&#x20;

```json
{
  "createTime": "2023-01-01T12:34:56.000Z",
  "message": "Intended amount violates risk limit for user",
  "code": "EXCEEDS_LIMIT_ACH"
}
```
